\Thm[Sylvester Criterion] A \(d\times d\) matrix is positive semi-definite if and only if all the upper left \(k\times k\) for \(k = 1,\ldots,d\) have a positive determinant.

Negative definite: \(det<0\) for all odd-sized minors, and \(det>0\) for all even-sized minors

Otherwise: indefinite.

\sep
